A high-end, premium developer portfolio built for **Ragul M**, featuring a dark-mode-first glassmorphism aesthetic, interactive 3D elements, and advanced animations.

## ✨ Key Features

- 💎 **Glassmorphism Design**: Modern, blurred background aesthetics with subtle borders and glow effects.
- 🍱 **Bento Grid Layout**: Visually stunning grid system for showcasing projects.
- 🪄 **Interactive Particles**: Custom canvas-based high-performance particle background.
- 🌐 **3D Elements**: Torus-knot geometry with Distorted Sphere effects using Three.js/Fiber.
- 🚀 **Mac-style Floating Dock**: Interactive spring-physics social bar for quick access.
- 🏗️ **Next.js 14 App Router**: Optimized performance with server-side components and static generation.
- ⚡ **Recruiter Fast-Track**: Dedicated section for quick resume downloads and profile access.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics**: [Three.js](https://threejs.org/) & [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Fonts**: [Google Fonts (Space Grotesk, Inter)](https://fonts.google.com/)

## 🚀 Getting Started

First, clone the repository:

```bash
git clone https://github.com/ragulhm/astral-architect-io.git
cd astral-architect-io
```

Install the dependencies:

```bash
npm install --legacy-peer-deps
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
